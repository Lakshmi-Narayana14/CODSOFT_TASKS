Personal Portfolio Website

CodSoft Frontend Development Internship – Task 1

This project was developed as Task 1 of my CodSoft Frontend Development Internship. It is a responsive Personal Portfolio Website designed to showcase my profile, skills, projects, education, achievements, and contact information. The website is built using HTML, CSS, and JavaScript with a clean, modern, and user-friendly interface.

✨ Features

- Responsive design for all devices
- Modern and clean user interface
- Home, About, Skills, Projects, and Contact sections
- Smooth navigation
- Project showcase with details
- Contact information for easy communication
- Simple and easy-to-customize code structure

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
